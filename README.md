# Introduction
This is a server monitoring tool designed for both personal and organizational use.

## Features
- Monitor one or multiple servers
- View current server status
- View server activity for the last 24 hours
- Server-specific logs
- Overload alerts
- Role-based access for admins and supports and clients

## Use Cases
- Personal server monitoring
- Small team infrastructure monitoring
- Organization-level server supervision

## Roles
- Admin
- Support
- Client
